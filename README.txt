Riverbend Community Health Center - Accessibility Demo Site

Purpose:
This mock site was created for accessibility assessment demonstrations using tools such as WAVE and Accessibility Insights.

How to use:
1. Unzip the folder.
2. Open index.html in Chrome or Edge.
3. Run the WAVE browser extension and/or Accessibility Insights.

Intentional accessibility issues included:
- Missing image alternative text
- Empty alternative text on a meaningful image
- Vague links such as "Read more" and "Click here"
- Duplicate link text with different destinations
- Form fields without visible labels
- Table without proper header cells
- Skipped heading level
- Low contrast text
- Raw URL used as link text

Note:
The images are local SVG assets included in the zip so the page will work without relying on external image URLs.
